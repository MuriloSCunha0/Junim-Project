# Prompt simples para conversão Delphi → Java Spring

Converta o código Delphi fornecido para Java Spring Boot seguindo estas diretrizes:

## MAPEAMENTOS BÁSICOS:
- TForm → @RestController
- TDataModule → @Service + @Repository  
- Classes business → @Service
- Queries SQL → JPA Repositories

## ESTRUTURA JAVA:
```
src/main/java/
├── controller/  # REST APIs
├── service/     # Lógica de negócio
├── repository/  # Persistência
├── entity/      # Entidades JPA
└── dto/         # Transferência de dados
```

## TECNOLOGIAS:
- Spring Boot 3.x
- Spring Data JPA
- Spring Web MVC
- Bean Validation

Gere código Java limpo e bem estruturado.
